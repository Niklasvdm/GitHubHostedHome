Adding N8N to the home lab

A walkthrough of deploying N8N — the open-source workflow automation tool — inside an LXC container on my Proxmox home lab.

Why N8N?

N8N provides a visual workflow builder that connects APIs, services, and scripts without vendor lock-in. Running it locally means full control over data and integrations.

LXC setup

Details on provisioning the LXC container, configuring networking, and installing N8N to run as a service.

Key takeaways

  • LXC containers offer a lightweight alternative to full VMs for single-service workloads.
  • N8N's self-hosted mode unlocks community nodes and unrestricted execution.
  • Integrating with existing home lab services (reverse proxy, DNS) was straightforward.

With N8N running, the next step is building automation workflows that tie together monitoring, alerts, and daily tasks.
